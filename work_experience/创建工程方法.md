# 创建工程方法

1. 更改java版本为1.7。
2. 配置公司私服。详见$MAVEN_HOME$/config/settings.xml。
    ```
    <profiles>
        <profile>
        <id>myProfile</id>

        <properties>
            <sonar.jdbc.url>jdbc:mysql://192.168.21.153:30002/sonar?useUnicode=true&amp;characterEncoding=utf8</sonar.jdbc.url>
            <sonar.jdbc.driver>com.mysql.jdbc.Driver</sonar.jdbc.driver>
            <sonar.jdbc.username>root</sonar.jdbc.username>
            <sonar.jdbc.password>password</sonar.jdbc.password>
            <sonar.host.url>http://192.168.21.153:30001/</sonar.host.url>
            <sonar.login>admin</sonar.login>
            <sonar.password>itframe@!QAZ</sonar.password>
        </properties>

        <repositories>
            <repository>
            <id>myRepository</id>
            <name>Repository for me</name>
            <url>http://192.168.100.10/nexus/content/groups/public/</url>
            </repository>
        </repositories>

        <pluginRepositories>
            <pluginRepository>
            <id>cmos-nexus-plugin</id>
            <name>cmos-nexus-plugin</name>
            <url>http://192.168.100.10/nexus/content/groups/public/</url>
            <releases>
                <enabled>true</enabled>
            </releases>
            <snapshots>
                <enabled>false</enabled>
            </snapshots>
            </pluginRepository>
        </pluginRepositories>
        </profile>
    </profiles>
    ```
3. 使用powershell打命令的话，-D参数周围要加双引号。
    ```
    mvn "-Dfile.encoding=UTF-8" "-DinteractiveMode=false" "-DgroupId=com.cmos" "-DartifactId=eoms" "-Dversion=1.0.0-SNAPSHOT" "-DarchetypeGroupId=com.cmos" "-DarchetypeArtifactId=itframe-web-archtype" "-DarchetypeVersion=1.1.1-SNAPSHOT" org.apache.maven.plugins:maven-archetype-plugin:RELEASE:generate
    ```
